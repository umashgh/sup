📦 Fresh Install

Full setup from scratch on a new machine (Raspberry Pi or similar).
Clone repo
git clone <repo-url> ~/apps/sup
Create virtualenv & install deps
cd ~/apps/sup && python3 -m venv .venv && .venv/bin/pip install -r requirements.txt
Run migrations
cd ~/apps/sup/sup_backend && ../.venv/bin/python manage.py migrate
Seed expense master data
cd ~/apps/sup/sup_backend && ../.venv/bin/python manage.py load_expense_master
Create admin user
cd ~/apps/sup/sup_backend && ../.venv/bin/python manage.py createsuperuser
Collect static files
cd ~/apps/sup/sup_backend && ../.venv/bin/python manage.py collectstatic --no-input
Install systemd user service
cp ~/apps/sup/scripts/sup-django.service ~/.config/systemd/user/ systemctl --user daemon-reload && systemctl --user enable --now sup-django
Install Cloudflare tunnel
curl -L --output cloudflared.deb https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-arm64.deb sudo dpkg -i cloudflared.deb
Then: cloudflared tunnel login → cloudflared tunnel create salaryfree → copy config to /etc/cloudflared/config.yml → sudo cloudflared service install
Set up Telegram alerts
mkdir -p ~/.config/salaryfree cat > ~/.config/salaryfree/telegram.conf << 'EOF' TELEGRAM_BOT_TOKEN=<token> TELEGRAM_CHAT_ID=<chat_id> EOF chmod 600 ~/.config/salaryfree/telegram.conf
Copy healthcheck timer: sf-healthcheck.service + sf-healthcheck.timer to ~/.config/systemd/user/, then systemctl --user enable --now sf-healthcheck.timer
Verify everything
curl -s https://salaryfree.in/health/ && echo "✅ all good"
